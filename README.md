Turning two messy real-world datasets — **FIFA 21 players** and **healthcare records** — into clean, analysis-ready tables using **Power Query (M)** in Excel.

## What I did
- Removed duplicates, blank and error rows
- Fixed inconsistent data types (numbers, dates, text)
- Standardised text: trimmed spaces, cleaned symbols, unified formatting
- Split and merged columns, renamed fields to a consistent standard
- Built repeatable queries so the cleaning re-runs automatically on new data

## Tools
Power Query (M) · Excel

## Files
- `fifa21_raw_data.csv` — raw FIFA 21 player data

- `healthcare_messy_data.csv` — raw healthcare data with quality issues
- `Healthcare_Data_Cleaning_Project.xlsx` — cleaned healthcare dataset with applied query steps
