# Очищення даних у Power Query — FIFA 21 та Healthcare

**🌐 Мова:** [English](README.md) · **Українська**

![Power Query](https://img.shields.io/badge/Power%20Query-M-217346?logo=microsoftexcel&logoColor=white)

Перетворення двох «брудних» реальних наборів даних — **гравці FIFA 21** та **медичні записи** — на чисті, готові до аналізу таблиці за допомогою **Power Query (M)** в Excel.

## Що я зробила
- Видалила дублікати, порожні та помилкові рядки
- Виправила невідповідні типи даних (числа, дати, текст)
- Стандартизувала текст: прибрала зайві пробіли й символи, уніфікувала формат
- Розділила та об'єднала стовпці, привела назви полів до єдиного стандарту
- Побудувала повторювані запити, щоб очищення автоматично застосовувалося до нових даних

## Інструменти
Power Query (M) · Excel

## Файли
- `fifa21_raw_data.csv` — сирі дані гравців FIFA 21
- `FIFA21_Data_Cleaning_Project.xlsx` — очищений набір FIFA 21 із кроками запиту
- `healthcare_messy_data.csv` — сирі медичні дані з проблемами якості
- `Healthcare_Data_Cleaning_Project.xlsx` — очищений медичний набір із кроками запиту
